USAGE                                                           *neovim-ghcid*

Run this command to start Ghcid in a terminal buffer.
>
	:Ghcid
<
When ghcid reports an error, use ":cc" to jump to it, and ":cn" to jump to the
next one.

To kill ghcid:
>
	:GhcidKill
<
